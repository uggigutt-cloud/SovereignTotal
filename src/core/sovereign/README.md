# Sovereign Modules Bundle

This zip contains the TypeScript modules and SQL schema produced in the conversation.

## Contents
- `db/` PGLite client wrapper
- `graph/` Graph builder (DB -> CaseGraphSnapshot)
- `engine/` Rule engine runner + attribute normalizer
- `rules/` Deterministic rulepack API + utilities + core rules
- `workers/` Web Worker for normalization + rule execution
- `ui/` Minimal helper to run rules via Worker
- `schema/` PGLite SQL schema (v1)

## Notes
- All IDs generated by the rule engine are deterministic (FNV-1a based).
- The worker expects a `PgLiteLike` instance to be initialized inside the worker or bridged in.
